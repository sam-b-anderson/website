# sambanderson.com

Personal site for Sam Anderson — professional front door, interactive work showcase, and playable about experience.

Built with: Next.js 16, Tailwind v4, Framer Motion, Zustand, shadcn/ui, deployed to Vercel.

## Development

```bash
npm install
npm run dev      # Start dev server (Turbopack)
npm run build    # Production build
npm run lint     # ESLint
```

## Structure

| Route    | Experience                          |
| -------- | ----------------------------------- |
| `/`      | Front door — professional landing   |
| `/work`  | Card pack experience                |
| `/about` | Machine experience                  |
